# Money Track App

## Deskripsi

Money Track App adalah aplikasi mobile sederhana yang digunakan untuk membantu pengguna dalam mencatat dan mengelola keuangan pribadi. Aplikasi ini memungkinkan pengguna mencatat pemasukan dan pengeluaran, menghitung saldo secara otomatis, serta menampilkan laporan keuangan sehingga kondisi keuangan dapat dipantau dengan mudah.

---

# Tujuan

* Membantu pengguna mengontrol keuangan harian maupun bulanan.
* Memudahkan pencatatan pemasukan dan pengeluaran.
* Menampilkan laporan keuangan secara sederhana.
* Menghitung saldo secara otomatis (real-time).
* Memberikan kemudahan dalam mengelola data akun pengguna.

---

# Fitur Utama

* Login pengguna
* Registrasi akun baru
* Input data pemasukan
* Input data pengeluaran
* Perhitungan saldo otomatis
* Laporan keuangan
* Manajemen profil
* Edit profil
* Logout

## Kelebihan Aplikasi

* Tampilan aplikasi sederhana dan mudah dipahami oleh pengguna.
* Proses login dan registrasi memudahkan pengelolaan akun pengguna.
* Dapat mencatat pemasukan dan pengeluaran dengan mudah.
* Saldo diperbarui secara otomatis setelah transaksi disimpan.
* Menampilkan laporan keuangan sehingga pengguna dapat mengetahui kondisi keuangannya.

## Penjelasan Tampilan

### 1. Splash Screen 1

Merupakan tampilan awal saat aplikasi dijalankan dengan latar belakang sederhana sebagai proses inisialisasi aplikasi.

### 2. Splash Screen 2

Menampilkan animasi transisi (slide/lipatan) sebelum pengguna diarahkan ke halaman berikutnya.

### 3. Intro Screen

Menampilkan logo Money Track beserta ucapan selamat datang kepada pengguna.

### 4. Loading Screen

Halaman yang digunakan untuk proses loading data serta pengecekan status login pengguna.

### 5. Landing Page

Halaman utama yang menampilkan logo Money Track dan tombol **Mulai** untuk menuju halaman Login atau Register.

### 6. Login

Halaman yang digunakan pengguna untuk masuk ke aplikasi dengan memasukkan email dan password.

### 7. Register

Halaman pendaftaran akun baru dengan mengisi nama lengkap, email, dan password.

### 8. Form Tambah Transaksi

Halaman yang digunakan untuk menambahkan data transaksi berupa pemasukan atau pengeluaran dengan mengisi jenis transaksi, nominal, dan keterangan.

### 9. Laporan Keuangan Versi 1

Menampilkan ringkasan total pemasukan, total pengeluaran, dan sisa saldo pengguna.

### 10. Laporan Keuangan Versi 2

Menampilkan laporan keuangan dengan tata letak yang berbeda sebagai alternatif tampilan.

### 11. Profil (View Mode)

Menampilkan informasi akun pengguna serta tombol **Edit Profil** dan **Logout**.

### 12. Profil (Edit Mode)

Halaman yang digunakan untuk memperbarui nama, email, dan password pengguna.

# Cara Menjalankan Aplikasi

1. Clone repository ini.
2. Buka project menggunakan Android Studio.
3. Sinkronkan Gradle.
4. Jalankan aplikasi menggunakan Emulator atau perangkat Android.
5. Lakukan registrasi akun jika belum memiliki akun.
6. Login menggunakan akun yang telah dibuat.
7. Tambahkan data pemasukan atau pengeluaran.
8. Lihat laporan keuangan dan kelola profil pengguna.

# Kesimpulan
